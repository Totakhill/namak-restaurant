# Namak Restaurant

Namak Restaurant is a premium responsive restaurant website inspired by Afghan hospitality, traditional cuisine, and modern dining experiences.

## Live Website

Live website:  
https://totakhill.github.io/namak-restaurant/

The website is deployed with GitHub Pages.

## Project Overview

This project was created as both:

- A professional restaurant website
- A hands-on learning project covering HTML, CSS, JavaScript, Git, GitHub, VS Code, Codex, responsive design, accessibility, and GitHub Pages

## Running Locally

No Node.js, npm, or build process is required because this is a static HTML, CSS, and JavaScript website.

1. Open `index.html` directly in a web browser.
2. Alternatively, open the project folder in VS Code and use a local development server if desired.
